# pytr (Modified)

This is a very slightly reengineered version of the [pytr](https://github.com/pytr-org/pytr) API with added functions and improved modules.

> **Disclaimer:**  
> This is an unofficial and reverse-engineered client for Trade Republic, provided for educational and research purposes only.  
> It is **not affiliated with or endorsed by Trade Republic**.

## Installation

```bash
pip install git+https://github.com/SoerenFox/pytr-Modified-.git
```

If the command `pytr` is not recognized after installation, add this to your PATH:

`%APPDATA%\Python\Python312\Scripts` on Windows or `~/.local/bin` on Linux/macOS.

## Usage

Easily accessible through the terminal. For further information, make use of the commands below or refer to the [original repository](https://github.com/pytr-org/pytr).

```bash
pytr
pytr command_name --help
```
